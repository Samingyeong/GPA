# 🎓 한밭대학교 GPA 계산기

한밭대학교 학점 체계를 기반으로 한 GPA(평균 학점) 계산기 웹 애플리케이션입니다.

## ✨ 주요 기능

- 📚 과목별 성적 입력 및 관리
- 🔍 **과목 자동완성 기능** - 한밭대학교 과목 목록에서 검색 및 선택
- 🎯 **자동 학점 입력** - 과목 선택 시 기본 학점 자동 입력
- 🧮 실시간 GPA 계산 (4.5 만점 기준)
- 📊 총 학점 및 과목 수 표시
- 🎨 모던하고 직관적인 UI/UX
- 📱 반응형 디자인 (모바일 지원)

## 🚀 시작하기

### 사전 요구사항

- Node.js 16.x 이상
- npm 또는 yarn

### 빠른 시작 (2단계)

#### 1단계: 백엔드 서버 실행

```bash
cd server
npm install  # 최초 1회만
npm run dev
```

**성공 시**: `🚀 Server running on http://localhost:3001`

#### 2단계: 프론트엔드 실행 (새 터미널)

```bash
cd ..  # 루트 디렉토리로
npm install  # 최초 1회만
npm run dev
```

**성공 시**: 브라우저가 자동으로 열립니다 (`http://localhost:3000`)

### 📚 상세 가이드

- **빠른 시작**: [QUICK_START.md](./QUICK_START.md)
- **테스트 가이드**: [TEST_GUIDE.md](./TEST_GUIDE.md)

## 📖 사용 방법

1. **과목 추가**: "과목 추가" 버튼을 클릭하여 새로운 과목을 추가합니다.
2. **과목 검색 및 선택**: 
   - 과목명 입력 필드에 과목명을 입력하면 자동완성 목록이 표시됩니다.
   - 키보드 화살표 키(↑↓)로 선택하고 Enter로 확인하거나, 마우스로 클릭하여 선택합니다.
   - 과목을 선택하면 기본 학점이 자동으로 입력됩니다.
3. **정보 입력**: 필요시 학점을 수정하고, 성적을 선택합니다.
4. **계산**: "학점 계산하기" 버튼을 클릭하여 GPA를 계산합니다.
5. **결과 확인**: 우측 결과 패널에서 평균 학점과 상세 정보를 확인합니다.

### 💡 자동완성 사용 팁
- 과목명의 일부만 입력해도 검색됩니다 (예: "프로그래밍" 입력 시 관련 과목 표시)
- 자동완성 목록에는 과목명, 학점, 카테고리가 표시됩니다
- 직접 입력도 가능하며, 자동완성 목록에 없는 과목도 추가할 수 있습니다

## 🎯 학점 체계

한밭대학교는 4.5 만점 기준의 학점 체계를 사용합니다:

| 등급 | 점수 | 설명 |
|------|------|------|
| A+   | 4.5  | 최우수 |
| A    | 4.0  | 우수 |
| B+   | 3.5  | 양호 |
| B    | 3.0  | 보통 |
| C+   | 2.5  | 미흡 |
| C    | 2.0  | 미흡 |
| D+   | 1.5  | 낙제 |
| D    | 1.0  | 낙제 |
| F    | 0.0  | 낙제 |

## 🔧 커스터마이징

### 학점 체계 변경

`src/utils/gpaCalculator.js` 파일에서 `gradePoints` 객체를 수정하여 다른 학점 체계를 적용할 수 있습니다.

```javascript
export const gradePoints = {
  'A+': 4.5,
  'A': 4.0,
  // ... 원하는 학점 체계로 수정
}
```

### 과목 데이터 업데이트

과목 데이터는 Excel 파일에서 자동으로 추출됩니다:

1. **Excel 파일 준비**: `book_csv/` 폴더에 한밭대학교 과목 정보가 담긴 Excel 파일(.xls 또는 .xlsx)을 넣습니다.
2. **데이터 추출 실행**:
   ```bash
   npm run parse-excel
   ```
3. **자동 생성**: `src/data/hanbatCourses.js` 파일이 자동으로 업데이트됩니다.

**참고**: Excel 파일은 다음 컬럼을 포함해야 합니다:
- `개설과목`: 과목명
- `학점`: 학점 수
- `이수구분`: 카테고리 (전필, 전선, 교필, 교선 등)
- `개설학과`: 학과명 (선택)
- `과목코드`: 과목 코드 (선택)

현재 포함된 데이터:
- **715개 이상의 실제 과목**
- **14가지 이수구분 카테고리** (일선, 전필, 전선, 교필, 교선 등)
- **모든 과목의 학점 정보**
